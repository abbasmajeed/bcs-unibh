INTRUÇÕES DE USO:

Para melhor utilização deste trabalho, utilize a versão 2012b
ou superior do software Matlab.

* Diretórios:

~/prt 	   : Este diretório contem algumas funções 
	     utilizadas para o desenvolvimento do trabalho.

~/database : Este diretório contem a base de dados
	     utilizada como base de conhecimento do algoritmo KNN.

~/installer: Pacote com os arquivos de instação em standalone aplication
	     OBS: Para utilizar a aplicação sem ter o matlab instalado é necessário instalar uma 
	     	  runtime do matlab, a qual nós ja disponibilizamos dentro do diretorio installer, 
	     	  é necessario rodar o instalador em modo de compatibilidade com windows xp, devido a algumas dll's do windows.
		  se não conseguirem instalar o MCR (Matlab Compiler Runtime) execute a funcao startup.m igual eu fiz na apresentação.



startup.m  : Função Principal do projeto

* Instalação

Para realizar a instalação do aplicativo desenvolvido é necessário instalar 
uma runtime do matlab, essa runtime esta dentro do pacote na pasta installer
e deve ser instalada primeiro.

OBS: Como este trabalho esta em uma versão beta, é aconselhavel a execução, 
do mesmo dentro do ambiente do matlab.
