  1. (H) HMM Biblioteca e Designer - Esta biblioteca permite que os modelos probabilísticos de sequência para ser construído através da utilização de modelos ocultos de Markov (HMM) e Markov hierárquica modelos HMMs (HHMMs) em OCaml linguagem de programação. [GPL]

  89. Árvore Visualizer - Software que permite navegar (voar), através da árvore de dados, aumentar o zoom em nós interessantes, clique em barras para obter contagens, e lugares interessantes marca na árvore. Inclui conjuntos de dados para automóveis, de voto, produzir e pesquisa médica. Usa LEDA, ([AFL] licenciados somente). [GPL]
  90. Máquina UCI Aprendizagem - Um repositório de bases de dados, teorias de domínio e geradores de dados que são utilizados pela comunidade de aprendizado de máquina para a análise empírica da máquina de algoritmos de aprendizado da Universidade da Califórnia em Irvine. [Livre]
  91. UMDHMM e outros programas estatísticos - Esta ferramenta implementa modelos ocultos de Markov e aplicação de parte de fala-tagging. Também disponível; um software de testes variados hipótese para dados gaussianos, e um groundtruth / edição de metadados e visualizar kit de ferramentas para OCR. [GPL]
  92. VIBES: Inferência Variacional para redes bayesianas - Um pacote de software que permite-variância modelada inferência posterior para ser executado automaticamente em uma rede Bayesiana. [GPL]
  93. Weka 3: Data Software Mining - Uma coleção de ferramentas que implementam árvores de decisão e tabelas, os alunos de regra, Naive Bayes, máquinas de vetor de suporte, votou perceptrons, perceptron multi-camada.Regimes de meta incluem ensacamento, empilhamento e reforço. Escrito em Java. [GPL]
  94. What If: Web-based Descoberta Científica - Um motor algoritmo que calcular tudo, desde a simetria, ângulos de torção, fração polar através da análise de proteínas e ângulos de ligação. Versão on line. [Livre]
  95. WinBUGS: Projeto BUGS - Um programa stand-alone para permitir práticas métodos MCMC disponíveis para os estatísticos aplicados. Ou um ponto de interface e clique pode ser usado para controlar a análise ou uma interface gráfica pode ser construída. O projeto BUGS também inclui links para GeoBUGS para análise espacial, modelagem farmacocinética para PKBUGS e OpenBUGS para desenvolvimentos mais recentes. [GPL]
  96. WinMine Toolkit - Um conjunto de ferramentas para Windows 2000/NT/XP que lhe permitem construir modelos estatísticos de dados. [Livre]
  97. aiParts - C + + classes para multi-decisão os problemas de aplicação da técnica de Hope-alta com programas de exemplo. [MIT]
  98. libarow - Um software e uma biblioteca enorme para a classificação implementado com a Regularização Adaptativa de vetores de pesos (AROW) algoritmo. [MIT License]
  99. libbpfl - A Biblioteca probabilidade Bayesian Filtering - Uma biblioteca de propósito geral para a filtragem Bayesiana escrito em C + +. [LGPL]
  100. mPOEMS em JAVA - uma estrutura genérica para os mPOEMS algoritmo evolutivo de busca. Ele foi projetado para resolver problemas de otimização, com um número ilimitado de objetivos. Este site fornece todas as fontes e algumas implementações de exemplares, por exemplo, do problema n rígido mochila. [AFL]
  101. mloss: linguagem de máquina Software Open Source - Uma lista esforço da comunidade de pesquisa reprodutível através do software de código aberto, livre acesso aos dados e resultados, e padrões abertos para intercâmbio. [FREE]
  102. shogun - Uma interface de objeto genérico SVN com muitas implementações, incluindo OCAS, Liblinear e LibSVM e SVMLight, SVMLin e GPDT. Cada SVN fornece implementações dos kernals mais comuns, como Ploynomial, Linear, Gaussian, e Kernel Sigmond. Implementado em C + + e interfaces para Matlab, R, Octave e Python. [GPL]
